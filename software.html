My Opinion

To make our works easy to reproduce, I release software related to my papers periodically in this page. I strongly believe this is the most important responsibility for every researcher. If you have any question about my software or experiment results, just drop me an email.

Software Release

  • PIAS: Practical Information-Agnostic Flow Scheduling
    Just a simple proof-of-concept prototype system for paper testbed experiments. I am still working on it. I also release evaluation scripts. For more details about PIAS project, please read our technical report.

  • ICTCP: Incast Congestion Control for TCP
    I implement ICTCP in Linux for our incast project. My implementation strictly follows the description of Haitao's CoNEXT paper. For more details about evaluation results, you can read our ICNP paper.

  • PAC: Proactive ACK Control
    I implement PAC as a Linux kernel module for our incast project. Our testbed evaluation shows that PAC outperforms DCTCP and ICTCP in incast congestion. For more details about evaluation results, you can read our ICNP paper.
